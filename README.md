# 🌍 BasmaWorld

**BasmaWorld.com** — A unified platform integrating three pillars:

| Pillar | Description |
|--------|-------------|
| 🎵 **BASMA Music Academy** | Gamified music school with Skill Tree, XP system & AI mentor Echo |
| 🏆 **Major Winners League (MWL)** | Content agency & marketing firm managing client leads |
| 🤝 **Hopes Chance** | Resource Navigator connecting youth 16-30 to housing, jobs & mental health |

## 🛠️ Tech Stack
- **Frontend**: Next.js 14 + Tailwind CSS
- **Hosting**: Vercel
- **Database**: Airtable
- **AI**: Hermes + Emergent + Claude
- **Automation**: GetViktor (Slack)
- **CRM**: HubSpot + Apollo.io
- **Analytics**: PostHog

## 🚀 Getting Started
```bash
npm install
npm run dev
```
